## Objective

ccxt 기반 범용 거래소 브로커(CcxtBroker)를 구현하여 Bybit/OKX의 OHLCV 데이터 수급과 주문 실행을 지원한다. 기존 BinanceBroker의 ccxt 패턴을 일반화하여 config/broker.json에 거래소를 추가하면 자동으로 해당 거래소가 활성화된다.

Purpose: 멀티거래소 지원으로 데이터 소스와 실행 경로를 다변화하여 단일 거래소 의존도를 줄인다.
Output: engine/execution/ccxt_broker.py, 수정된 broker_factory.py, tests/test_ccxt_broker.py
