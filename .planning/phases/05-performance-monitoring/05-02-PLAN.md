- StrategyPerformanceMonitor._handle_alerts(snapshots, session):
  - WARNING snapshot: notifier.send_performance_alert(snapshot) 호출

## Success Criteria

1. 20거래 윈도우 Sharpe/승률이 baseline 대비 15% 이상 하락 시 Discord WARNING embed가 발송된다
2. 30거래 롤링 Sharpe < -0.5 시 Discord CRITICAL embed가 발송되고 해당 전략이 자동 일시정지된다
3. 알림 embed에 전략명, 현재/기준 Sharpe, 저하율, 승률, 윈도우 크기가 포함된다
4. 한 전략의 CRITICAL이 다른 전략에 영향을 주지 않는다
5. 모든 테스트 통과, 기존 테스트 회귀 없음
