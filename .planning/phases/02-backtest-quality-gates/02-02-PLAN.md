  <name>Task 1: WalkForwardValidator 구현 + 테스트</name>
  <files>engine/backtest/walk_forward.py, tests/test_walk_forward.py</files>
  <behavior>
    - validate(equity_curve) returns ValidationResult with mode="walk_forward"
    - Default: n_windows=5, train_pct=0.7, gap_threshold=0.5 (CONTEXT.md locked)
    - 500-bar equity curve split into 5 windows, each 70% IS / 30% OOS
    - Each window produces WindowResult with is_sharpe, oos_sharpe, gap_ratio, passed
    - gap_ratio = oos_sharpe / is_sharpe (0 if is_sharpe == 0)
    - window.passed = gap_ratio >= 0.5
    - overall_passed = all windows passed
    - Monotonically increasing equity curve => all windows pass (high Sharpe both IS and OOS)
    - Random walk equity curve => likely fails some windows
    - Edge case: equity curve too short (< n_windows * 10) => raises ValueError
  </behavior>
  <action>
    1. `engine/backtest/walk_forward.py`:
       - WalkForwardValidator 클래스: __init__(n_windows=5, train_pct=0.7, gap_threshold=0.5)
       - validate(equity_curve: pd.Series) -> ValidationResult
       - OHLCV가 아닌 equity curve를 직접 분할 (RESEARCH.md Open Question 1: equity curve split 방식 — 단순하고 plan 단독 테스트 가능)
       - n = len(equity_curve); window_size = n // n_windows; train_size = int(window_size * train_pct); test_size = window_size - train_size
       - 각 윈도우: equity_curve[i*window_size : i*window_size+train_size] = IS, 나머지 = OOS
       - IS/OOS 각각 pct_change로 returns 계산 후 Sharpe 계산
       - _calc_sharpe(returns, periods=252) static method — returns.mean()/returns.std()*sqrt(periods), std==0이면 0.0
       - too short면 ValueError("equity curve too short for {n_windows} windows")
       - NOTE: skfolio WalkForward는 OHLCV split에 유용하지만, 여기서는 equity curve 분할이므로 직접 구현이 더 적합. skfolio는 plan 05(CPCV)에서 활용.

    2. `tests/test_walk_forward.py`:
       - 단조 증가 equity curve (np.linspace) -> 전체 PASS 테스트
       - 랜덤 워크 equity curve (seed=42) -> 일부 FAIL 테스트
       - too short equity curve -> ValueError 테스트
       - custom gap_threshold=0.0 -> 전체 PASS 테스트
       - 결과 mode == "walk_forward" 확인
       - windows 개수 == n_windows 확인
       - gap_ratio 계산 정확도 테스트
