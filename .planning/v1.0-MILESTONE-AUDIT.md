## Remaining Gaps

### HIGH: MTFConfirmationGate not injected from bootstrap (MON-04)

- **From:** Phase 6 `engine/strategy/mtf_filter.py`
- **To:** Phase 9 `engine/interfaces/bootstrap.py` → `TradingOrchestrator(mtf_filter=...)`
- **Impact:** MTF 필터가 프로덕션에서 항상 비활성. 단기 타임프레임 신호가 상위 TF 방향 확인 없이 통과.
- **Fix:** `build_trading_runtime()`에서 `MTFConfirmationGate` 생성 후 `TradingOrchestrator(mtf_filter=...)` 전달

### HIGH: PromotionGate missing backtest_repo in callers (LIFE-03)

- **From:** Phase 11 `PromotionGate(backtest_repo=...)`
- **To:** Discord `paper_trading.py` lines 222, 261; API `paper.py` lines 88, 125, 158
- **Impact:** backtest baseline Sharpe 비교가 프로덕션 승격 경로에서 항상 skip. 6개 기준만 평가.
- **Fix:** `PromotionGate` 생성 시 `backtest_repo=BacktestRepository()` 전달

### MEDIUM: DashboardDataService missing repositories (MON-03)

- **From:** Phase 3 `PaperRepository`, Phase 2 `BacktestRepository`
- **To:** Phase 8 `engine/interfaces/dashboard/data_service.py`
- **Impact:** 대시보드에서 Paper PnL 곡선, 백테스트 이력 비교 데이터 미표시
- **Fix:** `DashboardDataService`에 `PaperRepository`, `BacktestRepository` 의존성 추가
