# 세력계단 지표 (주식단테)

## 개요
EMA + 표준편차 기반 계단형 지지선 지표. 세력이 만드는 가격 계단을 시각화하여 매수 타이밍 포착.

## 구성 지표 (3개)

### 1. 계단 (메인)
```
a = shift(eavg(c,26) + 2.5 * StdDev((C+H+L)/3, 26), 25)
b = HIGHEST(a, 계단길이)
x = VALUEWHEN(1, a > b(1), a)
y = VALUEWHEN(1, a < a(1), x)
if(c*(100+계단탈출)/100 >= a and eavg(c,112) < eavg(c,448), a, y)
```

### 2. 근접1 (매수 준비 - 지원군 위에 있을 때)
```
z1 = (eavg(c,112) > c and IchimokuSpan2*(100+구름이격)/100 > c)
이평조건 = eavg(c,224)*(100+이격224)/100 >= c and eavg(c,224)*(100-이격224)/100 <= c
if(c*(100+ready)/100 >= a and a*(100+단기이격)/100 > c and eavg(c,112) < eavg(c,448) and z1 and 이평조건, a, null)
```

### 3. 근접2 (매수 준비 - 지원군 아래에 있을 때)
```
z2 = (eavg(c,112) < c and IchimokuSpan2*(100+구름이격)/100 > c)
if(c*(100+start)/100 >= a and a*(100+단기이격)/100 > c and eavg(c,112) < eavg(c,448) and z2 and 이평조건, a, null)
```

## 변수

| 변수명 | 설정값 | 타입 | 설명 |
|--------|--------|------|------|
| 이평방법 | 지수이평 | 이평 | EMA |
| 계단길이 | 7 | 정수형 | HIGHEST 룩백 |
| 지원군 | 112 | 정수형 | 중기 EMA 기간 |
| ready | 8 | 정수형 | 근접1 진입 이격(%) |
| start | 5 | 정수형 | 근접2 진입 이격(%) |
| 단기이격 | 8 | 정수형 | 계단 대비 단기이격 |
| 계단탈출 | 8 | 정수형 | 계단 돌파 기준(%) |
| 이격224 | 8 | 정수형 | 224 EMA 이격 필터 |
| 구름이격 | 8 | 정수형 | 일목 구름 이격 |

## 핵심 로직
- EMA(112) < EMA(448) = 하락 추세 확인 (세력 매집 구간)
- 계단선 = EMA(26) + 2.5σ를 25봉 시프트 → 세력의 매집 상한선
- 가격이 계단에 근접하면 근접1/2 시그널 발생
- 일목균형표 구름과 224 EMA 이격도를 추가 필터로 사용

## 변환 불가 사유
- VALUEWHEN, HIGHEST 커스텀 로직이 ta-lib에 없음
- 일목균형표 센코우스팬2 조건 필요
- shift(시프트) 함수 필요
