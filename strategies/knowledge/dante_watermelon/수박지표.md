# 수박지표 (주식단테)

## 개요
EMA + 표준편차 기반 매수 타이밍 지표. 껍질(외곽선) 안에 수박(빨간 막대)이 차면 매수 신호.

## 구성 지표 (2개)

### 1. 껍질지표 (파란색 막대)
```
n1 = shift(eavg(c,26) + 2.5*StdDev((C+H+L)/3, 26), 25) / 크기1
y1 = 0
가 = eavg(c,112)*(100+이평간이격)/100 < eavg(c,224)
    and eavg(c,224)*(100+이평간이격)/100 < eavg(c,448)
    and eavg(c,224)*(100+단기이격)/100 >= c
    and eavg(c,112) < eavg(c,성함락)
x1 = if(가, n1, y1)
if(eavg(c,112) > c, y1, x1)
```

### 2. 수박지표 (빨간색 막대)
```
값 = 고가/크기1
y2 = 0
x2 = if(조건, if(n1 < 값, 고가/크기2, 값), y2)
if(eavg(c,112) > c, y2, x2)
```

## 변수

| 변수명 | 설정값 | 타입 | 설명 |
|--------|--------|------|------|
| 기간 | 26 | 정수형 | EMA/StdDev 기간 |
| 이평방법 | 지수이평 | 이평 | EMA |
| d1 | 2.5 | 실수형 | 표준편차 배수 |
| 크기1 | 7.5 | 실수형 | 껍질 스케일링 |
| 크기2 | 4.5 | 실수형 | 수박 스케일링 |
| 불꽃 | 고가 | 가격 | 수박 상한 기준가 |
| 이평간이격 | 1 | 정수형 | EMA 간 이격(%) |
| 성함락 | 5 | 정수형 | 112 EMA 대비 가격 위치 |
| 단기이격 | 15 | 정수형 | 224 EMA 이격 필터(%) |

## 핵심 로직
- EMA(112) < EMA(224) < EMA(448) = 정배열 하락 확인
- 가격이 112 EMA 아래일 때만 활성화
- 껍질 = 볼린저밴드 상한의 스케일링 값 (매집 상한)
- 수박 = 고가 기반 모멘텀. 껍질보다 작을 때 빨간 막대 확대
- 수박이 껍질을 채울수록 매수 타이밍 근접

## 업그레이드 vs 오리지널
- **오리지널**: EMA 배열만 확인 (112>224 or 224>448)
- **업그레이드**: 이평간이격, 단기이격, 성함락 조건 추가 → 더 엄격한 필터

## 변환 불가 사유
- shift() 함수, 커스텀 스케일링 로직
- 시각화 지표(막대 채움)로 매매 신호가 아닌 참고 지표
