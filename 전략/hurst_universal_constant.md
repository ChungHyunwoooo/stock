# Hurst 지수 — 시장의 보편 상수

## 1. Hurst 지수란

### 원래 유래
1951년 영국 수문학자 Harold Edwin Hurst가 나일강 범람 패턴을 연구하며 발견.
"올해 범람이 크면 내년에도 클 확률이 높다" — 자연현상의 장기 기억(long memory).

### 수학적 정의
시계열의 **추세 지속성**을 0~1로 측정하는 지수.

```
H = 0.5: 랜덤워크 (동전 던지기). 과거가 미래에 영향 없음.
H > 0.5: 추세 지속. 올랐으면 계속 오를 확률이 높음. (persistent)
H < 0.5: 반추세. 올랐으면 내릴 확률이 높음. (anti-persistent)
```

### 직관적 이해
```
H = 0.5 (랜덤워크):
  가격: ─╱╲─╱╲╱─╲╱╲─╱╱╲─   방향이 매번 독립 (동전 던지기)

H = 0.75 (추세 지속):
  가격: ─╱╱╱╱╱╱──╲╲╲╲╲─╱╱   한번 방향 잡으면 유지되는 경향

H = 0.3 (반추세):
  가격: ╱╲╱╲╱╲╱╲╱╲╱╲╱╲╱╲   오르면 바로 내리고 내리면 바로 오름
```

### 측정 방법 (R/S 분석)
```
1. 시계열을 길이 n인 구간으로 나눔
2. 각 구간에서:
   R = 누적 편차의 범위 (max - min)
   S = 표준편차
3. R/S를 n에 대해 로그-로그 회귀
4. 기울기 = H

R/S ∝ n^H
log(R/S) = H × log(n) + c
```

## 2. 왜 중요한가

### 시장 효율성 가설과의 관계
```
EMH(효율적 시장 가설): 시장은 랜덤워크 → H = 0.5
현실: H ≈ 0.75 → 시장은 랜덤워크가 아님

→ 추세가 지속되는 경향이 있다
→ 모멘텀 전략이 작동하는 수학적 근거
→ "어제 올랐으면 오늘도 오를 확률이 50%보다 높다"
```

### 매매 전략과의 연결
```
H > 0.5 (실제 시장):
  추세추종 전략이 기대값 양(+)
  모멘텀 팩터가 존재하는 이유
  "추세는 친구다"의 수학적 근거

만약 H = 0.5였다면:
  모든 기술적 분석이 무의미
  어떤 전략도 동전 던지기와 같음
  시장 타이밍은 불가능

만약 H < 0.5였다면:
  평균회귀 전략만 유효
  추세추종은 음의 기대값
```

## 4. 왜 0.75인가 — 해석

### 인간 행동의 구조적 특성
```
H = 0.75의 의미:
  "추세가 지속되지만 영원하지는 않다"
  = 모멘텀 + 유한한 기억

  인간의 판단:
    최근 오르면 → "더 오를 것" (모멘텀 추종) → H > 0.5
    하지만 영원히 추종하지 않음 (적응, 이익실현) → H < 1.0

  이 균형점이 ~0.75
```

### ABM 시뮬레이터에서의 확인
```
에이전트 기반 시뮬레이션으로 확인:
  momentum_weight=0.6, adaptation_decay=0.9 → H ≈ 0.82
  momentum_weight=0.0, mean_reversion=0.5  → H ≈ 0.60

  모멘텀 추종 에이전트가 있으면 H > 0.5
  적응적 전략 전환이 있으면 H < 1.0

  → 시장 참여자의 "추세 추종 + 적응"이 H ≈ 0.75를 만듦
  → 주식이든 크립토든 참여자가 인간이면 동일한 H
```

### Lo의 적응적 시장 가설과의 연결
```
Andrew Lo (2004):
  "시장 효율성은 고정이 아니라 진화한다"

  H = 0.75가 시기별로 안정적 =
  개별 전략의 수익성은 변해도 (기술적 분석 수익 하락 등)
  시장의 추세 지속성 자체는 변하지 않음

  → 참여자가 바뀌어도, 도구가 바뀌어도, 시대가 바뀌어도
    "인간이 거래하는 한" H ≈ 0.75
```

## 5. 실전 함의

```
1. 추세추종 전략은 수학적으로 유효하다
   H > 0.5 → 모멘텀 팩터가 존재 → 추세추종의 기대값 > 0
   (단, 실행 비용, 슬리피지, 과적합 제외)

2. 순수 평균회귀는 단기에만 유효하다
   H > 0.5이므로 장기적으로는 추세 방향이 유리
   평균회귀는 "추세 내 눌림목"에서만 양의 기대값
   = 앵커(추세) 방향으로의 회귀만 유효

3. 타임프레임은 중요하지 않다
   15분이든 일봉이든 H ≈ 0.75
   → 어떤 타임프레임에서든 추세 지속성은 동일
   → 타임프레임 선택은 H와 무관, 거래 비용/생활패턴의 문제

4. 종목 선택도 H에 영향 없다
   BTC든 SPY든 PEPE든 H ≈ 0.75
   → 종목별 차이는 변동성(std), 꼬리(kurtosis)에서 나타남
   → 추세 지속성 자체는 종목 무관
```

## 6. 한계와 주의

```
1. H는 "확률"이지 "확실"이 아님
   H = 0.75 ≠ "75% 확률로 같은 방향"
   장기적 통계적 경향일 뿐, 개별 봉은 여전히 불확실

2. H는 "방향"을 알려주지 않음
   "추세가 지속된다"는 것이지 "어느 방향으로"는 별개
   방향 판단은 여전히 지표/L3/앵커 추정이 필요

3. H가 변할 수 있는가?
   2024~2026 데이터에서는 안정적 (0.74~0.80)
   하지만 AI 트레이딩 비율이 극단적으로 높아지면?
   HFT가 지배하면 H가 0.5에 수렴할 가능성
   → 주기적 재측정 필요
```
